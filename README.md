# Schema-Driven UI Generator

<a name="english"></a>
## English

A full-stack demonstration project that automatically generates functional CRUD (Create, Read, Update, Delete) interfaces by consuming JSON Schemas directly from a .NET 9 backend. This project compares multiple industry-standard libraries to evaluate their integration with the **Refine** framework.

### 🚀 Key Features

- **Multi-Library Comparison**: Compare two powerful approaches for schema-to-form generation:
    - **Refine + RJSF**: High-fidelity forms using `react-jsonschema-form`.
    - **Refine + Uniforms**: Highly customizable and performant forms using `uniforms` + `ajv`.
- **Auto UI Generation**: Utilizes .NET 9 `JsonSchemaExporter` to convert C# entity models into JSON Schema, enabling automatic form generation without manual coding.
- **Modern Data Management**: Integrated with **Refine**, providing a robust data provider layer, notification system, and routing.
- **Dynamic Schema Exploration**: Built-in `SchemaFetcher` tool to inspect raw JSON schemas generated by the backend in real-time.
- **Intelligent Relational Support**: Automatically handles foreign key relationships (e.g., Categories for Products) by mapping them to searchable Select components.
- **Organized Architecture**: Cleanly separated component structure for better maintainability and scalability.

### 🛠 Tech Stack

#### Backend
- **Framework**: .NET 9 (ASP.NET Core)
- **ORM**: Entity Framework Core 9.0
- **Database**: PostgreSQL
- **Schema Export**: `System.Text.Json.Schema` (JsonSchemaExporter)

#### Frontend
- **Framework**: [Refine](https://refine.dev/) (Enterprise-grade CRUD framework)
- **UI Kit**: Material UI (MUI) v6
- **Form Libraries**:
    - [react-jsonschema-form (RJSF)](https://rjsf-team.github.io/react-jsonschema-form/)
    - [Uniforms](https://uniforms.tools/) (with AJV validation)
- **Data Fetching**: TanStack React Query v5, Axios
- **Build Tool**: Vite

### 📂 Directory Structure (Frontend)

```text
src/
├── components/
│   ├── common/       # Shared utilities (SchemaFetcher)
│   ├── rjsf/         # RJSF specific components (AutoManager, AutoForm)
│   └── uniform/      # Uniforms specific components (UniformEntityManager)
├── App.jsx           # Main entry point with library selection
└── Refine...Example  # Page containers for each implementation
```

<a name="한국어"></a>
## 한국어

백엔드(.NET 9)에서 제공하는 JSON Schema를 직접 소비하여 CRUD(생성, 조회, 수정, 삭제) 인터페이스를 자동으로 생성하는 풀스택 데모 프로젝트입니다. 이 프로젝트는 여러 산업 표준 라이브러리를 **Refine** 프레임워크와 통합하여 비교 분석합니다.

### 🚀 주요 기능

- **라이브러리 비교 분석**: 두 가지 강력한 스키마 기반 폼 생성 방식을 비교할 수 있습니다:
    - **Refine + RJSF**: `react-jsonschema-form`을 사용한 고충실도(High-fidelity) 폼 구현.
    - **Refine + Uniforms**: `uniforms`와 `ajv`를 사용한 고성능 및 커스텀 최적화 폼 구현.
- **자동 UI 생성**: .NET 9의 `JsonSchemaExporter`를 사용하여 C# 엔티티 모델을 JSON Schema로 변환하고, 이를 기반으로 코딩 없이 폼을 자동 생성합니다.
- **현대적인 데이터 관리**: **Refine** 프레임워크를 기반으로 강력한 데이터 프로바이더, 알림 시스템, 라우팅 기능을 제공합니다.
- **동적 스키마 탐색**: 백엔드에서 생성된 원본 JSON 스키마를 실시간으로 확인할 수 있는 `SchemaFetcher` 도구가 내장되어 있습니다.
- **지능형 관계 지원**: 외래 키 관계(예: 상품의 카테고리)를 자동으로 감지하여 검색 가능한 Select 컴포넌트로 매핑합니다.
- **체계적인 아키텍처**: 유지보수와 확장성을 고려하여 컴포넌트를 기능별(RJSF, Uniforms, Common)로 구조화했습니다.

### ⚙️ Getting Started / 실행 방법

#### 1. Database Setup
Run PostgreSQL using the provided `docker-compose.yml` in the root directory.
```bash
docker-compose up -d
```

#### 2. Running the Backend
- **Visual Studio**: Set `ReactFormAutoGenerate.Server` as the startup project and press **F5**.
- **CLI**: 
  ```bash
  cd ReactFormAutoGenerate.Server
  dotnet run
  ```

#### 3. Running the Frontend
```bash
cd ReactFormAutoGenerate.Client
npm install
npm run dev
```
